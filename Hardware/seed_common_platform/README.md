# Daisy Seed Common Platform

This hardware project implements a eurorack compliant breakout board for the daisy seed

## Features
- 8 potentiometer controls 
- momentary PB
- 3 POS Switch x 2
- multiplexed 
- configurable jacks

### Configurable Jack IO: 
- stereo audio IO
- DAC outputs x 2
- CV inputs x 8
- gate output x 2
